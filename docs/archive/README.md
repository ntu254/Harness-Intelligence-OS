# Archive

This folder preserves historical planning documents that explain how HI-OS
evolved. Archived files are not the current operating entrypoint.

Current users should start with:

- `README.md`
- `docs/HARNESS.md`
- `docs/FEATURE_INTAKE.md`
- `docs/CONTEXT_RULES.md`
- `docs/COMMAND_COOKBOOK.md`

Archived phase plans and early specs remain useful for provenance, but they
should not override current decisions, schemas, story packets, or validation
evidence.
